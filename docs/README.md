# citation docs

citation の公式ドキュメント

`v1.4.0` 以前に使用されていた `m2en/citation-docs` から移行したものになります。

## Command Reference

- build: `mkdocs build`
  - ドキュメントのビルド
- dev: `mkdocs serve`
  - ドキュメントを開発者モードで表示
